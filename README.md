# Grocery Store Management System

A Django-based Grocery Store Management System designed to streamline inventory management, billing, supplier management, employee tracking, and daily store operations.

## Features

* Product Management
* Inventory Tracking
* Supplier Management
* Billing System
* Employee Management
* Authentication & Authorization
* PostgreSQL Database Integration

## Tech Stack

* Python
* Django
* PostgreSQL
* HTML
* CSS
* JavaScript

## Installation

```bash
git clone https://github.com/codewithjuhili/grocery-store-management-system.git
cd grocery-store-management-system
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
```
